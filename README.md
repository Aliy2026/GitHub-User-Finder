# GitHub User Finder – Поиск пользователей GitHub

**Автор:** Алия Байсултанова

## Описание
Приложение с графическим интерфейсом (Tkinter) для поиска пользователей GitHub через официальное GitHub REST API.  
Возможности:
- Поле ввода для поиска по логину
- Отображение результатов поиска в виде списка
- Добавление пользователей в избранное (с сохранением в JSON)
- Удаление из избранного
- Открытие профиля пользователя в браузере
- Проверка: поле поиска не может быть пустым

## Как использовать API
Приложение использует публичный GitHub API без аутентификации:
- Эндпоинт: `https://api.github.com/search/users?q={query}&per_page=20`
- Лимит: 60 запросов в час с одного IP (для неавторизованных). Для тестирования достаточно.
- При желании можно добавить токен в заголовок `Authorization`, изменив метод `search_users`.
